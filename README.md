# bsl_html_code_editor

Редактор кода 1С для встраивания в поле HTML документа на базе WebKit 605.1.11

## Описание

Одностраничный HTML-файл с редактором кода для языка 1С (BSL). 
Использует технологию textarea с невидимым слоем для подсветки синтаксиса.

## Структура проекта

```
/workspace
├── README.md           # Этот файл
└── src/
    └── editor.html     # Основной файл редактора
```

## Функционал

1. **Нумерация строк** - отображение номеров строк слева от редактора
2. **Подсветка синтаксиса** - поддержка следующих элементов:
   - Ключевые слова 1С (если, иначе, функция, процедура и др.)
   - Строки (в кавычках)
   - Комментарии (начинающиеся с //)
   - Числа
   - Директивы препроцессора (начинающиеся с &)
3. **Контекстная подсказка** - вызывается сочетанием клавиш `Ctrl+Space`

## Использование

### Встраивание в WebKit компонент

```html
<iframe src="editor.html" style="width: 100%; height: 100%; border: none;"></iframe>
```

### Горячие клавиши

- **Tab** - вставка табуляции
- **Ctrl+Space** - показать контекстную подсказку для ключевого слова

## Технические детали

- Шрифт: Consolas/Monaco, 14px
- Высота строки: 1.5
- Отступы: 8px со всех сторон, 48px слева для нумерации строк
- Размер табуляции: 4 пробела

## Совместимость

- WebKit 605.1.11 и выше
- Используется только vanilla JavaScript (без внешних зависимостей)
